# Quest

A simple to do list application that lets me add, edit and delete using firebase as the backend

## How to get started

Clone Repository and install all packages
```
npm install
```
Run the application
```
npm run start
```
Start using the application

## Technologies Used:

* React.js
* Redux
* Firebase
